# Gestor de Empleados - Serialización Java

## Descripción
Aplicación en Java que demuestra la serialización de objetos para gestionar una lista de empleados. Permite guardar objetos Empleado en un archivo binario y recuperarlos posteriormente.

## Funcionalidades
- Crear objetos Empleado (nombre, id, salario)
- Guardar colecciones de empleados en archivos binarios
- Leer y recuperar empleados desde archivos binarios
- Visualizar los datos de los empleados recuperados

## Tecnologías
- Java
- Serialización (interfaz Serializable)
- ObjectOutputStream / ObjectInputStream
- FileOutputStream / FileInputStream
- ArrayList
- Manejo de excepciones IOException y ClassNotFoundException
